Usage instructions:  

to use visit [caffeinated.qzz.io/musepod](https://caffeinated.qzz.io/music-player.html)
it comes preloaded with canned heat by jamiroquai  
but if you wanted to upload your own songs you can upload up to 3 .mp3 files using the choose files button in the top left corner  

how i made it:  

i used kenny yip coding's [tutorial](https://www.youtube.com/watch?v=APjb5Er03UE) on making a music player to start as i didnt know any html, css or js prior  
after completing the tutorial i added looping and shuffling, i then added the ability to import your own songs and some cool ui  

what i hope to add in future:

a vb.net app that could connect
